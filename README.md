# 前端同学注意

~~你们写的网页文件按都放在src/main/resources/static目录下面，后面我们的接口会不断的更新到这个文档里面。~~

明确一下确定要求：

1. 像css/js/img/font这些文件放在static目录下面

2. html文件放在tmplates目录下面

3. 登录首页面叫index.html

4. 访问地址：localhost：8080
